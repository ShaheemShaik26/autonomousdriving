# Dataset Attribution

This repository supports Kaggle vehicle-detection data ingestion, training, and evaluation.

When you use the vehicle-detection dataset or benchmarks, keep the attribution below and comply with the original license terms.

```bibtex
@dataset{kaggle_vehicle_detection,
  title = {Vehicle Detection Dataset},
  author = {Killa92},
  year = {2024}
}
```

License summary:

- Attribution is required.
- Follow the Kaggle dataset card and any upstream license terms included with the dataset.
- Review redistribution rights before publishing derivative datasets.
